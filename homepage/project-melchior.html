Toekomstig project: Melchior mascotte

Dit project is een volledig persoonlijk en zelfstandig project om mijn skills te zetten met het programma illustrator. In dit project hoop ik een originele mascotte te ontwerpen. De mascotte zal gebaseerd worden op een dier, in dit geval is het een vos omdat ik dit meer vind passen met het bedrijf. Deze mascotte zal in een recht staande positie staan, hij zal een drankje vast hebben en een shirtje dragen van het bedrijf. Ik hoop na mijn examens met dit project verder te gaan. En deze achteraf toe te voegen aan mijn portfolio. Ik ben gestart met dit project om mij zelf te testen op het gebruik van de pen tool, het gebruik maken van kleuren, patronen maken en schaduw effecten toevoegen. Het is een creatief project gebaseerd op design.
