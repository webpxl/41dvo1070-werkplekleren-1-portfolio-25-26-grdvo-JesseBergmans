Project: scripting

Je kan op de afbeeldingen klikken om deze te vergroten. Dit project is mijn eerste grote project dat ik heb gemaakt voor mijn studie. In dit project is het de bedoeling om een website op te bouwen voor een fictie bedrijf. Op deze site moet er een homepage voorzien zijn met een gerecht van de dag aangetoond aan het begin van de pagina. Hiernaast moesten we ook een pagina hebben met de volledige menu van mijn bedrijf, je moet in dit menu de optie hebben om jou producten te verschuiven volgens prijs en je moet voorzien dat je producten kan toevoegen aan je winkelmand. Hier staat ook een maximum op per product. De website werd opgebouwd met gebruik van HTML, CSS en javascript.
