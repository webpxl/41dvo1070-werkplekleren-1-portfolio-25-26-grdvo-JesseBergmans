Projectbeschrijving:

         In dit project gaan wij voor ons als persoon een portfolio opbouwen met gebruik
         van HTML, CSS en javascript. In deze portfolio vind je terug hoe ik ben tijdens
         en na het werk, hoe je mij kan contacteren en welke projecten ik heb gemaakt.
         De focus van het project is voornamelijk gebaseerd op technische kennis en visueel design.


Doelstellingen:

- Korte voorstelling van wie je bent en wat jou speciaal maakt.
- Projecten die je gemaakt hebt of aan deel genomen hebt ( minstens 3).
- Weetjes ( een aanvulling aan je project in mijn geval de over mij pagina).
- Alleen gebruik maken van HTML, CSS en javascript ( geen andere talen of wordpress).

gebruikte technieken en software:

- Webstorm (HTML, CSS en Javascript)
